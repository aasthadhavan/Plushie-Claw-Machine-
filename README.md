<h2>Plushie Claw Machine 🎀</h2> 
A cute little browser-based claw machine game where you try your luck at grabbing adorable plushies! Inspired by the arcade claw machines that somehow always seem to have a mind of their own. 

<h3>Features</h3>
- Cute claw machine gameplay  
- Interactive controls  
- Smooth animations  
- Fun arcade-inspired experience  
- Runs directly in your browser — no installation needed!


<h3>Built With</h3>
- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript


<h3>Play the Game</h3>
Simply open the live website or run locally:

```bash
git clone https://github.com/aasthadhavan/Plushie-Claw-Machine-.git
cd Plushie-Claw-Machine-
```

Then open:

```
index.html
```

in your browser.

<h3>Objective</h3>
Move the claw, time it carefully, and see if you can collect the cutest plushies before luck runs out!
*(Disclaimer: The claw's success rate may or may not resemble real arcade machines...)
